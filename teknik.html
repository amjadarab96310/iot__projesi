Mikroişlemci Yazılım Modeli

Projenin yazılım mimarisi Raspberry Pi üzerine kurulmuştur. Python dili ile OpenCV kütüphanesi kullanılarak yüz tanıma algoritması geliştirilmiştir.

Kullanılan Donanım Bileşenleri

  • Raspberry Pi 3 Model B
  • Kamera Modülü (Everest Hd 480p Usb Mikrofonlu Pc Kamera Web Kamerası Sc-825)
  • Kapı Kilidi (CDM Selenoid Kapı Kilidi - 12v - 53x22x42mm)
  • 3 Adet 18650 3.7v 6800 Mah Şarjlı El Feneri Powerbank Pili
  • Tek Kanallı Röle (5V)
